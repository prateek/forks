# forks

Prateek's daily-driver forks: upstream plus my open PRs, kept installable and
current. This repo is both the fleet monorepo and its Homebrew tap.

```sh
brew tap prateek/forks https://github.com/prateek/forks
brew install prateek/forks/<tool>-fork
```

How it works and how to operate it: [AGENTS.md](AGENTS.md).
